# 3D VRM Model Viewer with Animation

## Project Overview
This project is an interactive 3D viewer built with **Three.js** and **Vanilla JavaScript**, allowing users to load and interact with **VRM models**.  

Key features include:
- Smooth **animations** on VRM models  
- **User interaction** with mouse controls  
- Dynamic **scene setup** including lighting and environment  
- **Camera controls** for immersive experiences  
- Responsive, real-time rendering for browser-based 3D visualization  

The goal is to provide a lightweight and customizable platform to visualize animated 3D characters directly in the browser.

## Tech Stack
- **Vite** – Fast development and build tool  
- **Vanilla JavaScript** – Core programming language  
- **Three.js** – 3D rendering library  
- **vrm** – Loader and utilities for VRM models  
- **OrbitControls** – Camera controls for user interaction  
- **GLTFLoader** – Loading VRM/GLTF models  
- **HTML & CSS** – Basic page structure and styling
